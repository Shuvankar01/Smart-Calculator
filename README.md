# Smart Calculator Pro

A modern **Scientific Calculator Web App** built using **HTML, CSS, and JavaScript**.

This project demonstrates frontend development skills including UI design, event handling, and mathematical operations.

---

## 🚀 Features

* Basic Calculator Operations (+, −, ×, ÷)
* Scientific Calculator Mode
* Keyboard Input Support
* Calculation History
* Clear History Option
* Dark / Light Theme Toggle
* Animated UI Buttons
* Mobile Responsive Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
